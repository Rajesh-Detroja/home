Getting Started with R

Installation

The first thing need to do to get started with R is to install it on a computer. R works on pretty much every platform available, including the widely available Windows, MacOS X, and Linux systems.

  • Install R on Windows

  • Install R on MacOS

  • Install R on UBUNTU

There is also an integrated development environment available for R that is built by RStudio. This IDE—it has a nice editor with syntax highlighting, there is an R object viewer, and there are a number of other nice features that are integrated. RStudio can be installed as follows:

  • Install RStudio

  • Website: RStudio
